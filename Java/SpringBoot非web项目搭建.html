    # Spring Boot 非web项目搭建

    TIP

    之前一直搭建的是springboot + maven + web 的项目，坐flink demo的时候，尝试了一下非web项目

    # 1、 在启动页面选择Create New Project后，会出现如下

    TIP

    选择Maven项目后，勾选 Create from archetype。对于非web项目，选择maven-archetype-quickstart 能快速建立一个普通maven项目

    # 2、 下一步Next之后

    TIP

    填写groupid,artifaceId；再下一步，在Properties中添加 archetypeCatalog=internal。不加这个参数，在maven生成骨架的时候将会非常慢，有时候会直接卡住

    # 3、 新建项目名称，项目名称与项目保存的位置的文件夹的名称一致。所以可以只写project location就行

    # 4、 打包

    TIP

    生成meta-inf文件夹，选择file-project structure

    # 5、 选择artifacts

    # 6、 选择JAR->From modules with dependencies

    WARNING

    注意这边默认带\main\java，文件夹，我们直接放到src文件夹下面

    WARNING

    在pom文件中增加打包插件（注意：这边打包用命令打包，mvn package）

    <build>
    <plugins>
      <plugin>
        <groupId>org.apache.maven.plugins</groupId>
        <artifactId>maven-assembly-plugin</artifactId>
        <version>2.5.5</version>
        <configuration>
          <archive>
            <manifest>
              <mainClass>com.itbin.flink.App</mainClass>
            </manifest>
          </archive>
          <descriptorRefs>
            <descriptorRef>jar-with-dependencies</descriptorRef>
          </descriptorRefs>
        </configuration>
        <executions>
          <execution>
            <id>make-assembly</id>
            <phase>package</phase>
            <goals>
              <goal>single</goal>
            </goals>
          </execution>
        </executions>
      </plugin>
    </plugins>
  </build>
